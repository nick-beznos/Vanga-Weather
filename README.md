# Vanga-Weather

Its a test app, that uses OpenWeatherMap API and shows current weather of city, that you've searched. 
Features: 
- 100% programatic UI
- Dark mode friendly
- Accessibility Titles
- Animations (UITableViewDiffableDataSource)
